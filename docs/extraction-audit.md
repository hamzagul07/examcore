### Sample A — Paper 1 MCQ (`qp_12.pdf`) — **inspected**

**Source:** Gemini 2.5 Flash PDF analysis, 2026-06-06.

| Field | Observation |
|-------|-------------|
| Pages | ~16 |
| Question numbering | Arabic numerals `1`, `2`, … `40` (flat, no sub-parts) |
| Max nesting depth | 1 |
| Marks | Total marks on cover; **no per-question mark tags** |
| MCQ layout | A/B/C/D columns; some options in **tables** (Q3, Q37, Q39) |
| Diagrams | **High density** — ~19 figures per paper; CRO traces, force diagrams |
| Math | High density; dedicated **Data & Formulae** page (page 2) |
| Figure refs | "The diagram shows…", "The graph shows…" |

**Excerpt (Q14):**
> The force resisting the motion of a car is proportional to the square of the car's speed. The magnitude of the force at a speed of 20.0 m s⁻¹ is 800 N. What useful output power is required from the car's engine to maintain a steady speed of 40.0 m s⁻¹?

**Splitter implications:**
- Treat each integer question as a leaf; no `(a)(i)` nesting.
- MCQ options must be captured as structured choices, including table layouts.
- Formulae sheet is a separate non-question block (skip or store as metadata).
- Diagram extraction is critical (~19 per paper × 44 papers ≈ **800+ figures** for P1 alone).

**Mark scheme format (from `investigation-report.json`, `ms_11`):**
```
1 C  2 B  3 A  4 D  5 B ...
```
Answer-key only — `extracted_mark_points` will be 1 point per question (the correct letter).

## 6. LLM model policy (Gemini only)

All pipeline and production inference uses **paid Gemini API** (`GEMINI_API_KEY`). Model routing is centralized in `lib/ai/gemini-models.ts`:

| Model | Tasks |
|-------|-------|
| **`gemini-2.5-pro`** | Lesson generation (Prompt B), diagram descriptions, validation coverage checks, syllabus extraction (Phase 5), worked solutions |
| **`gemini-2.0-flash`** | Topic tagging (Phase 6), marking, OCR, segmentation, mark-scheme PDF extraction, chat, JSON-shaped structured output |

**PDF extraction:** Mathpix remains primary for math/LaTeX/layout (~$30 for 9702). Gemini Pro is secondary for diagram alt-text and pages where Mathpix confidence &lt; 0.85 (`MATHPIX_LOW_CONFIDENCE_THRESHOLD`).

Use `modelForTask('topic-tagging')` or pass `task` / `model` via `generateGeminiText()` — see `lib/ai/gemini-text.ts`.
